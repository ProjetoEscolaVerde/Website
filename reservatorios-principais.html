Sistema dos reservatórios principais


O sistema dos reservatórios principais visa principalmente utilização eficiente da água da chuva. Se não houver mais água disponível das chuvas é acionado a abastecimento por água tratada.

Este sistema possui dois sensores ultrassônicos à prova de água para o monitoramento da quantidade de água em cada reservatório (R1 e R2). Além disso, possui três válvulas solenoides 12V (V1,V2,V3) que são normalmente fechadas para o controle do fluxo de água e uma bomba de água 12V para bombear a água para os reservatórios secundários.

Como a bomba de água fica no primeiro reservatório (R1) e a válvula V2 é normalmente fechada, a água da chuva inicialmente enche o R1. Se o R1 se encontrar próximo a sua capacidade máxima, a válvula V2 é aberta para drenar parte da água do R1 para o R2. Este controle permite que a bomba de água fique submersa e funcione normalmente mesmo com níveis mais baixos de água.

As válvula V3 e V2 são aberta se o nível de água dos reservatórios ficarem muito baixos, o que permite que a água tratada proveniente do colégio Cândido portinari abasteça o R1, evitando falta de água para a irrigações. Se os dois reservatórios se encontrarem no limete da capacidade por causa de muita chuva a válvula V1 e V2 são abertas para liberar água dos reservatórios.

TIPOS DE GRÁFICOS

Diário

Os dados são coletados a cada exatos 30 minutos no intervalo fechado de 00h00m até 23h30m. Os dois primeiros gráficos se encaixam nesta categoria.

Mensal

O nível de água em cada dia é calculado pela média dos 48 valores coletados no respectivo dia (24 horas).

Anual

Dividido em semanas - média do nível de água dos 7 dias - e mostra todos os dados coletados deste o início das coletas. Consideramos a primeira semana de cada mês como a primeira semana iniciada no domingo em determinado mês.
