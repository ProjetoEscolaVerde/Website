Gerações Escol@Verde

Objetivo

Através das gerações anuais é possível a persistência do projeto Escol@Verde no Colégio Cândido Portinari juntamente com o progresso contínuo através das novas gerações. Todos os anos novos estudantes podem entrar no projeto para se juntar com aqueles que resolveram participar por mais um ano. É dever dos estudantes do ano anterior transmitir o que foi aprendido e mostrar o que foi feito nos anos anteriores para os novos integrantes, de forma a garantir que os novos estudantes consigam se inserir nos grupos específicos de pesquisa.
